# 📌 Student Grading System (.NET 10 Console App)

## 🚀 Overview
A console application that calculates student performance using loops, switch pattern matching, enums, and static members.

## 🎯 Objective
- Practice looping and array handling
- Use enums for menu-driven programs
- Implement switch pattern matching
- Understand static variables and methods

## 🛠️ Tech Stack
- Language: C#
- Framework: .NET 10
- Type: Console Application

## 🧠 Concepts Covered
- Arrays & Loops
- Static members
- Enums
- Switch Pattern Matching
- LINQ (Min, Max, Average)

---

## ⚙️ How It Works
1. User enters student name and 5 subject marks
2. Menu is displayed using enum options
3. Based on user choice:
   - Aggregate → Average marks
   - MinMark → Lowest marks
   - MaximumMark → Highest marks
   - Grade → Based on average
4. Switch pattern matching determines grade
